Detailed Description

Class used to draw stored objects.

The object to draw are divided by layers. Each layer have specific position based on previous and next layers. It can be presented as a list. Each layer can be drawn independently, from a range of layers or all layers can be drawn.

A manager can only have a predetermined number of instances (default is 1, cannot be changed).
