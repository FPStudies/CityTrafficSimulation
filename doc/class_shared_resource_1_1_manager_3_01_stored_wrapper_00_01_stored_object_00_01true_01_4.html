Detailed Description

template<typename StoredWrapper, typename StoredObject>
class SharedResource::Manager< StoredWrapper, StoredObject, true >

Manager that manage the shared resource. The shared resource have this property that it will not be deleted if some other object retains ownership of the resource. Resource can be deleted only if no other objects does not share ownership on it.
